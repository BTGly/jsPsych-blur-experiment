# 认知行为实验 1.2 —— 被试操作指南

## 打开实验

在浏览器（Chrome 或 Edge）打开实验链接：

```text
https://btgly.github.io/cognitive-behavior-experiment-1.2/
```

> 建议使用全屏模式，实验过程中关闭其他程序通知。

---

## 第一步：填写参数

| 字段 | 说明 |
|------|------|
| **被试编号** | 格式为 **S** + 三位数字（如 S001），由主试提供 |
| **练习次数** | 默认 24，一般无需修改 |
| **起始组 / 结束组** | 默认 1–11。已有实验记录的被试，保持默认即可，系统会自动从下一轮继续 |
| **上传授权码** | 请询问主试 |

---

## 第二步：实验流程

```
欢迎 → 练习 → 预实验 → 正式实验 → 结束
```

### 1. 练习（24 题）

- 屏幕上短暂显示一张图片，判断是"正常样本"还是"缺陷样本"
- **F 键** = 正常样本（数字 3）
- **K 键** = 缺陷样本（数字 8）
- 按住时长代表自信程度：越确定按得越久（最长 1 秒），松开即确认
- 每题后有对错反馈

### 2. 预实验（180 题）

- 首次参加时必须完成，用于测量个人感知曲线
- 3 组 × 60 题，无反馈
- 图片呈现 0.2 秒，请集中注意力
- **2 秒内必须按键**，超时视为未作答

### 3. 正式实验

- 共 11 轮，每轮 100 题
- 无反馈，请根据第一感觉作答
- 每轮结束后可休息

---

## 分次完成建议

不要一次性做完 11 轮。建议每次 3 轮左右：

| 次数 | 起始组 | 结束组 | 说明 |
|------|--------|--------|------|
| 第 1 次 | 1 | 3 | 预实验 + 1-3 轮 |
| 第 2 次 | 4 | 6 | 系统会自动从下一轮开始 |
| 第 3 次 | 7 | 9 | |
| 第 4 次 | 10 | 11 | |

> 保持默认 1–11 即可，系统会自动跳过已完成的轮次。

---

## 按键说明

| 按键 | 功能 |
|------|------|
| **F** | 选择"正常"（数字 3） |
| **K** | 选择"缺陷"（数字 8） |
| 按住不放 | 表达自信心（越长越自信，最长 1 秒后自动确认） |
| **Enter** | 确认/继续（文字界面使用） |
| **Esc** | 随时退出实验，已作答数据自动保存并上传 |

---

## 注意事项

1. 全屏模式下请关闭其他程序通知，避免弹窗干扰
2. 保持坐姿稳定，眼睛与屏幕距离保持一致
3. 请勿在疲劳、饮酒后参加实验
4. 按 Esc 可随时退出，数据不会丢失
5. 再次进入时使用同一编号，系统会自动从下一轮继续
